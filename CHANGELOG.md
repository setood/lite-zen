# Changelog

## [0.1.0] - 2026-04-18

### Added

- Toggle all UI panels with a single hotkey (`Cmd+K Cmd+\` / `Ctrl+K Ctrl+\`)
- Escape to restore panels when focused in the editor
- Hides: primary sidebar, bottom panel, activity bar, status bar, auxiliary sidebar
- Per-component hide/restore settings
- Workspace-scoped state persistence across VS Code restarts
